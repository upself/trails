Asset Management Tools Scheduled Outages



A scheduled maintenance release will be occurring on the dates noted below. Please be aware of this outage.

Deployment Period

Friday 9:00 pm EST - Saturday 9:00 am EST

The asset management technical services team will be implementing a standard weekly change window that will allow for database and application maintenance and performance adjustments to be completed.

Over the last year the data loaded to the infrastructure has contiued to increase as well as an increase in the number of users accessing this information through the related applications. As a step to improve performance of the applications and make adjustments due to this increase in volume the support team will define a change window during off-peak hours.

This will potentially impact user access to the applications in the list below. It is not always required for the application to be down during this window each week.

  • Customer Number Database (CNDB)
  • ATP
  • TRAILS 4
  • BRAVO
  • HART
  • Microsoft License Wizard
  • Asset Tag Generator
  • Signature Bank
