Please note that the 'Error/Warning Event Instruction Codes Mapping List' is used for you to search 'Event Handling Instruction' based on 'Event Handling InstructionCode'.

Event Instruction Code Definition Format: E/W(Error/Warning)-Event Group Code(3 chars)-Event Type(3 chars)-Sequence Number(3 chars)
For example: E-TBS-CRS-001
