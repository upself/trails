軟體搜尋說明

Search for Software：您可在此搜尋欄位內，輸入要尋找相關資訊的軟體名稱。 您可以輸入全部或部分要搜尋的軟體名稱。您輸入的名稱越明確，所得到的結果也就會越少越精確。 相反地，您輸入的名稱越普通，所得到的結果也就越一般。此欄位會識別空格。

您也可以按一下 View the Software Catalog 鏈結，設定進階搜尋。

 

軟體種類說明

Software Name：即包含產品製造商名稱的敘述名稱。

Manufacturer：即建立應用程式代碼的公司。

Software Category：即介定軟體的一般種類。

軟體編目檔表說明

Software Name：即包含產品製造商名稱的敘述名稱。

Level：軟體為可授權、元件或非可授權。可授權軟體要求使用者付款，以使用或執行軟體。元件則為某可授權軟體產品中的一部分套裝軟體。通常元件不會脫離可授權的母項軟體進行安裝，亦不需要自身的採購授權。非可授權的軟體通常為免付費軟體，不管作為任何用途都無須付費購買該軟體。

Manufacturer：即建立應用程式代碼的公司。

Filters：這個鏈結可將您導向特定軟體的過濾器清單。

Signatures：這個鏈結可將您導向特定軟體的簽章清單。

Editor：前次編輯編目檔之項目人員的 Intranet ID（例如，IBMer@us.ibm.com）。

軟體過濾器清單

Status：如果顯示綠色勾號，則表示過濾器為作用中。如果顯示紅色 X 號，則表示過濾器為非作用中。

Filter Name：過濾器的名稱。

Filter Version：過濾器的版本。

Mapped Version：套裝軟體的版本。

Last Editor：前次上傳軟體簽章之人員的 Intranet ID（例如，IBMer@us.ibm.com）。

Additional Comments：顯示任何其他資訊。

儲存庫帳號

帳號可分為連線與斷線模式兩種類型。連線類型帳號為即時帳號，並與 Signature Bank 直接連線。斷線模式帳號並未與 Signature Bank 直接連線，且需手動執行掃描。

連線儲存庫帳號說明

Status：綠色勾號表示此為作用中帳號。紅色 X 號表示帳號為非作用中。

Name：帳號名稱。

Type：即掃描技術或工具類型。

Version：掃描技術或工具的版本號碼。

Database Types：目前僅支援與 DB2 資料庫的完整自動連線。

Database Version：DB2 資料庫的版本。

Authenticated Version：指示掃描基礎架構是否包含鑑別資料。

Synchronize Signature：指示客戶是否希望接收「知識庫」所傳送的資料。

Editor：前次編輯此項目之人員的 Intranet ID（例如，IBMer@us.ibm.com）。

Last Edited：前次編輯檔案的日期與時間。

 

更新連線的儲存庫帳號說明

Bank Account Name：帳號名稱。

Bank Account Description：說明此帳號的意義。

Bank Account Type：欲使用的掃描技術或工具類型。

In the Bank Account Version：欲使用的掃描技術或工具的版本號碼。

Data Type：帳號為 ATP、Inventory 或 Baselines

Database Type：選取 DB2。 目前僅支援與 DB2 資料庫的完整自動連線。

Database Version：欲使用的 DB2 資料庫版本。

Database Name：DB2 資料庫名稱。

Database Schema：DB2 資料庫結構名稱。

Database IP：DB2 資料庫的 IP 位址。

Database Port：DB2 資料庫連線埠號。

Database User Name：DB2 使用者名稱。

Database Password：DB2 密碼。

Socks Connection Required：指示是否需要 Socks 連線。

SSH Tunnel Connection Required：指示是否需要 SSH 通道連線。

Tunnel Port：即以整數表示的通道埠號。

Authenticated Data：指示掃描基礎架構是否包含鑑別資料。

Synchronize Signatures：指示客戶是否希望接收「知識庫」所傳送的資料。

Status：指示帳號為作用或非作用中。
